🗺️ Product Roadmap

Four phases from MVP to market leadership

Q2 2026

Phase 1: MVP Launch

  • Gmail OAuth production setup
  • SMS reading (Android companion / SMS forwarding)
  • Real carrier API integration (Israel Post tracking)
  • Onboarding flow optimization
  • Push notifications (FCM)
  • App Store listing (PWA + TWA for Android)
  • Beta testing with 50 users
🎯 50 beta users
Q3-Q4 2026

Phase 2: Growth

  • WhatsApp message parsing
  • Barcode/QR scanning
  • Multi-carrier API (FedEx, DHL, BOX lockers)
  • Smart reminders ("Package expires tomorrow!")
  • Family sharing (household tracking)
  • User analytics dashboard
  • Marketing: social media, influencer partnerships
🎯 5,000 active users
Q1-Q2 2027

Phase 3: Monetization

  • PickPick Premium — unlimited family, priority notifications
  • AI delivery time predictions
  • Price comparison for shipping
  • Business API for eCommerce stores
  • Carrier partnerships (branded tracking pages)
🎯 50,000 users · Revenue start
H2 2027

Phase 4: Expansion

  • AI-powered delivery predictions
  • Package consolidation recommendations
  • Return tracking & management
  • Marketplace integration (price alerts)
  • Expand to Hebrew/Arabic speaking markets
  • iOS native app
🎯 200,000 users

⚠️ Risks & Mitigations

Identified risks with clear mitigation strategies

📱

SMS access requires app permissions

→ Gmail-first approach, SMS as Phase 2 via companion app

🔌

Carrier API access (Israel Post)

→ Web scraping fallback, negotiate API access in parallel

📉

Low initial adoption

→ Viral sharing features, family invites, referral program

🏃

Competition enters market

→ First mover advantage, superior UX, deep local integration
